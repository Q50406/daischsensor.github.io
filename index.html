ROS Native Support

Publish standard sensor_msgs/Imu and temperature out of the box. Includes custom interfaces for Euler angles and dynamic gyro bias.

Reference Architectures

Validated configurations for popular SLAM and Odometry packages. Copy-paste ready for your launch files.

Visual-Inertial (VIO)

High-frequency IMU data (200Hz+) is critical for VINS to recover scale and handle rapid motion blur.

VINS-Mono ORB-SLAM3
# vins_config.yaml
imu_topic: "/imu/data"
acc_n: 0.08
gyr_n: 0.004

LiDAR-Inertial (LIO)

Provides tight-coupling for LiDAR motion de-skewing and robust localization in feature-poor environments (tunnels).

LIO-SAM FAST-LIO2
# params.yaml
imuTopic: "/imu/data"
imuAccNoise: 3.99e-03
imuGyrNoise: 1.56e-03
extrinsicRot: [1, 0, 0, ...]

Sensor Fusion (EKF)

Fuse wheel odometry with IMU gyro/accel to prevent drift. Standard setup for AGVs and indoor mobile robots.

robot_localization nav2
# ekf.yaml
imu0: /imu/data
imu0_config: [false, false, false,
true, true, true, ...]
two_d_mode: true
From Zero to Data

Quick Start Guide

Clone, build, and run the node in under 3 minutes. Our drivers are designed to be lightweight and compile cleanly in your existing workspace.

bash — ROS 2 (Humble / Foxy)
# 1. Create workspace & clone repositories
$mkdir -p ~/ros2_ws/src && cd ~/ros2_ws/src
$git clone https://github.com/DaischSensor/im1r_ros2_interface.git
# 2. Install dependencies & build
$cd ~/ros2_ws
$rosdep install --from-paths src --ignore-src -r -y
$colcon build
# 3. Grant port permission & Launch Node
$sudo chmod 666 /dev/ttyUSB0
$ros2 run im1r_ros2_driver im1r_driver_node --ros-args -p serial_port:=/dev/ttyUSB0 -p baud_rate:=115200
bash — ROS 1 (Noetic / Melodic)
# 1. Create workspace & clone repository
$mkdir -p ~/catkin_ws/src && cd ~/catkin_ws/src
$git clone https://github.com/DaischSensor/im1r_ros1_driver.git
# 2. Build the workspace
$cd ~/catkin_ws
$catkin_make
# 3. Grant port permission & Run Node
$sudo chmod 666 /dev/ttyUSB0
$rosrun im1r_ros_driver daisch_im1r_node /dev/ttyUSB0 115200

Prerequisite: Ensure you have the C++ serial package installed before building. The easiest way is using rosdep: rosdep install --from-paths src --ignore-src -r -y.
